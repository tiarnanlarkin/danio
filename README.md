# Aquarium Hobby App

This is the code repo for the Aquarium Hobby App.

## Goals (MVP)
- Android first
- Login required
- Freshwater only for v1
- Logs (water tests/events/photos)
- Equipment as first-class objects (maintenance schedules + history)
- Tasks + reminders
- Units toggle (metric/imperial)
- AI identification for plants + fish with confidence gating
- Monetisation: subscription (Google Play), no Play "free trial" offer

## Repo layout
- `contracts/` — JSON schemas + (later) OpenAPI
- `apps/` — mobile app source (Flutter or native Android)
- `services/` — backend services (AI gateway, sync)
- `docs/` — architecture, decisions
