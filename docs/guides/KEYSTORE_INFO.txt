Danio Android signing - local-only setup
========================================

Current security status (2026-07-15): NOT RELEASE-READY.

The previously tracked signing values were removed from the current Git tip.
Public Git history retains the earlier exposed values. Do not use this file as a
credential source and do not sign or upload a release until Play Console App
Integrity and release history have been reviewed.

Keep the real configuration only in:
  apps/aquarium_app/android/key.properties

That file and private keystore formats are ignored by Git. Never commit them.
Use placeholders when documenting the format:

  storePassword=<YOUR_STORE_PASSWORD>
  keyPassword=<YOUR_KEY_PASSWORD>
  keyAlias=<YOUR_KEY_ALIAS>
  storeFile=<ABSOLUTE_OR_ANDROID_RELATIVE_KEYSTORE_PATH>

The local keystore and key.properties are intentionally not changed by this
containment. Whether the exposed material was ever registered with or used for
a Play app is unresolved and requires a separate Play Console review and user
decision before any rotation, reset, or history-remediation action.
