## 📝 Content Creation Guidelines

### **Lesson Structure Standard**
Each lesson should have:
- **Title**: Catchy, clear (max 6 words)
- **Description**: One-sentence hook
- **Duration**: 4-6 minutes reading time
- **Word Count**: 400-600 words
- **Sections**: 6-10 sections using varied types
- **Quiz**: 5-10 questions with explanations

### **Section Types to Use**
- `heading` - Break up content
- `text` - Main teaching content
- `keyPoint` - Critical takeaway (highlight)
- `warning` - Danger/common mistake
- `tip` - Helpful advice
- `funFact` - Engagement/curiosity
- `bulletList` - Easy scanning
- `numberedList` - Step-by-step instructions

### **Quiz Question Quality**
✅ **Good Questions:**
- Test understanding, not memorization
- Real-world scenarios
- Common misconceptions
- "Why" questions over "what" questions
- Varied difficulty

❌ **Avoid:**
- Trick questions
- Overly technical jargon
- "All of the above" (unless genuinely teaching)
- Questions answered only in quiz, not lesson

### **Tone & Style**
- **Conversational** - Write like talking to a friend
- **Encouraging** - Mistakes are learning opportunities
- **Practical** - Real-world examples
- **Honest** - Don't oversimplify or sugarcoat
- **Humorous** - Where appropriate (fun facts, warnings)
