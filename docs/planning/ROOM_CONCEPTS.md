# 🏠 Aquarium App - Room Theming & UX Concepts

> *"Your aquarium hobby, room by room"*

## Design Philosophy

The Aquarium App uses a **cozy house metaphor** where each room represents a different aspect of the fishkeeping hobby. The house belongs to an enthusiastic aquarist — every room shows signs of their passion, from fish-themed decorations to actual tanks scattered throughout.

### Unifying Elements (Present in ALL Rooms)
- **Warm wood textures** - Oak floors, wooden trim
- **Ambient water sounds** - Subtle, never intrusive
- **Natural lighting** - Warm daylight / soft lamplight at night
- **Small aquatic touches** - A fish ornament here, aquatic plant there
- **The "Through Line"** - A small 5-gallon betta tank visible through doorways, connecting spaces
- **Color palette anchor** - Teal/aqua accents appear in every room

### Day/Night Mode Philosophy
- **Day Mode**: Bright, productive, energetic
- **Night Mode**: Warm lamp glow, relaxed, meditative (tanks glow softly)

## 🚪 Navigation & Room Transitions

### Main Navigation Metaphor
The Living Room is the **hub**. All rooms connect through doorways/arches visible in the background.

```
                    ┌──────────┐
                    │  STUDY   │
                    │    📚    │
                    └────┬─────┘
                         │
    ┌──────────┐    ┌────┴─────┐    ┌──────────┐
    │ FRIENDS  │────│  LIVING  │────│ WORKSHOP │
    │    👥    │    │   ROOM   │    │    🔧    │
    └──────────┘    │    🛋️    │    └──────────┘
                    └────┬─────┘
                         │
    ┌──────────┐         │         ┌──────────┐
    │  TROPHY  │─────────┴─────────│  SHOP    │
    │    🏆    │                   │  STREET  │
    └──────────┘                   └──────────┘
```

### Transition Animation Principles
1. **3D spatial movement** - Camera physically moves between rooms
2. **Through doorways** - Never teleport, always walk through a door
3. **Sound bridges** - Audio from destination room fades in before visual
4. **The connecting tank** - That small betta tank is visible during transitions

### Transition Duration
- **Quick tap**: 0.3s fast transition
- **Intentional browse**: 0.8s scenic transition
- **First time**: 1.5s full reveal with ambient buildup
