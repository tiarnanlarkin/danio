## Privacy Policy Hosting — Recommendation

### ✅ Use GitHub Pages (Recommended)

**Why:** You already have a GitHub account and the codebase is there. GitHub Pages is free, deploys in under 5 minutes, and gives you a stable `https://` URL that Play Store accepts.

**How to set it up:**

1. Create a new public GitHub repository: `danio-privacy` (or add to your existing repo under a `docs/` folder on a `gh-pages` branch)

2. Create a file `index.html` (or `privacy.html`) with the privacy policy HTML content

3. In the repo settings → Pages → set source to `main` branch, `/docs` or `/root`

4. Your URL will be: `https://[yourusername].github.io/danio-privacy/`

**Best practice:** Create a dedicated repo `danio-app-legal` with:
- `index.html` → redirects to `/privacy`
- `privacy/index.html` → Privacy Policy
- `terms/index.html` → Terms of Service (for future use)

**Final URL to submit to Play Console:**
```
https://[yourusername].github.io/danio-app-legal/privacy/
```

**Alternative if you want a custom domain later:** Point `legal.danioapp.com` to GitHub Pages. Zero cost if you already own the domain.

## IARC Content Rating — Questionnaire Answers

The IARC (International Age Rating Coalition) questionnaire is completed inside Google Play Console under **Store Presence → Content Rating**. Here are the recommended answers for Danio:

Select: **Utility / Productivity / Educational content**

| **Sensitive topics or controversial content** | **No** | |
| **Designed for children (under 13)?** | **No / Mixed audience** | COPPA-compliant but not exclusively targeted at under-13s |

### Expected Rating Output

|---|---|

| ESRB (North America) | **Everyone (E)** |

**IARC overall rating: Everyone / All Ages** ✅

### Notes on AI Features
The IARC questionnaire may ask specifically about AI/chatbot features. Select "Yes" and specify it is **content-filtered, educational-only AI** (not a general-purpose chatbot, no user-to-user communication). This keeps the rating at Everyone — it does not push the rating up.

### Notes on "Designed for Children"
Do **not** select "Designed for children under 13" — this triggers COPPA-only mode in Play Store and restricts some features. Instead, select **"Mixed audience"** or leave unchecked. The app is COPPA-compliant in its data handling without needing the "designed for children" flag.
