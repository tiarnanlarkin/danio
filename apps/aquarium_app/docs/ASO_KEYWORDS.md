## Overview

Danio operates at the intersection of two Play Store categories:
1. **Education** — learning apps, spaced repetition, structured courses
2. **Lifestyle / Hobbies** — pet care apps, aquarium tools, fish identifier

Most competitor apps sit in one category or the other. Danio's positioning is strongest when it owns both simultaneously — we're not just a "fish care app" and not just a "learning app." We're the app that teaches you fishkeeping. That dual positioning is the keyword opportunity.

## A/B Test Plan — Title + Short Description

### Hypothesis
The current title ("Danio: Learn Fishkeeping") is strong but "Learn" front-loads educational positioning. Some users searching for a tank manager may skip past it. Test whether different positioning expands reach without hurting conversion.

---

### Title Tests

**Control (current):**
```
Danio: Learn Fishkeeping
```

**Variant T1 — Tool-forward:**
```
Danio: Aquarium App & Fish ID
```
*(29 chars ✅)*
*Tests whether "Aquarium App" + "Fish ID" pulls more tool-seeking users*

**Variant T2 — Community/hobby forward:**
```
Danio: Fishkeeping & Fish ID
```
*(28 chars ✅)*
*Drops "Learn," tests if removing the education frame increases download rate in tool searches*

**Recommendation:** Run Control vs T1 first. If T1 wins on installs but loses on D7 retention (because tool-seekers aren't the right user), revert. If both installs AND retention hold, T1 is the new control.

---

### Short Description Tests

**Control (Variant A from listing):**
```
Learn fishkeeping with 84 lessons, AI Fish ID & 125+ species guides.
```

**Test SD1 — Lead with AI (trending feature hook):**
```
AI Fish ID + 84 fishkeeping lessons & 125+ species care guides. Free.
```
*(70 chars ✅)*

**Test SD2 — Lead with "free" trust signal:**
```
Free fishkeeping app: 84 lessons, AI Fish ID & 125+ species guides.
```
*(67 chars ✅)*

**Test SD3 — Question hook:**
```
New to fishkeeping? 84 lessons, AI Fish ID & 125+ species guides. Free.
```
*(71 chars ✅)*

**Evaluation metrics:**
- Primary: Store listing conversion rate (impressions → installs)
- Secondary: D1 retention rate (are the right users downloading?)
- Test duration: minimum 2 weeks per variant, minimum 500 impressions per variant

**Recommendation order:** Run Control → SD1 → SD2. SD3 only if first two underperform.
